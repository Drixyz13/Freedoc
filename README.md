# 💉 FreeDoc – Book Your Doctor Online

Welcome to **FreeDoc**, a Rails-based learning project inspired by Doctolib. This app manages doctor-patient appointments in a multi-city, multi-specialty environment. 🏥

## 🧠 Learning Objectives

- Build and connect multiple models using Active Record
- Understand and implement **1-N** and **N-N** relationships (`has_many`, `belongs_to`, `has_many :through`, `has_and_belongs_to_many`)
- Create and seed a clean and coherent database
- Use the Rails console to manipulate real data

```bash
git clone [your_repo_url]
